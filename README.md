# 📦 Vault Experience

> O cofre definitivo do colecionador geek. Uma plataforma moderna de e-commerce e troca de artigos colecionáveis criada especialmente para o público nerd.

---

## 📌 Sobre o Projeto

O **Vault Experience** é um ecossistema digital projetado para entusiastas da cultura pop, cinéfilos, gamers e otakus. Mais do que uma simples loja virtual, a plataforma funciona como um "cofre" (Vault) onde usuários podem encontrar itens raros para comprar e também negociar/trocar colecionáveis repetidos ou antigos com outros membros da comunidade.

### ✨ Principais Funcionalidades

*   **🛒 E-commerce Geek:** Vitrine completa de produtos dividida por universos (Marvel, DC, Star Wars, Animes, Games, etc.).
*   **🔄 Sistema de Trocas (Trade Vault):** Área dedicada para usuários anunciarem colecionáveis que desejam trocar com outros colecionadores.
*   **📱 Interface Gamer/Dark Mode:** Layout moderno com cores vibrantes e contrastes projetados para agradar o público nerd, inspirado em launchers de jogos e plataformas de streaming.
*   **⭐ Sistema de Avaliação:** Um sistema na qual o usuário poderá fazer uma avaliação do produto e do vendedor. 


---

## 🛠️ Tecnologias Utilizadas

O projeto foi estruturado com as seguintes tecnologias e ferramentas:
*   🌐 **HTML5 Semântico** – Organização limpa e acessível dos componentes da loja e do sistema de trocas.
*   🎨 **CSS3 Avançado** – Design responsivo (Flexbox/Grid), uso de gradientes futuristas de alta qualidade e estilizações em Dark Mode.
*   ⚡ **JavaScript (ES6+)** – Responsável pela comunicação assíncrona com o ecossistema do backend, manipulação dinâmica do DOM e controle de fechamento de abas.
*   🟢 **Node.js** – Ambiente de execução do servidor e construção da API RESTful do ecossistema.
*   🍃 **MongoDB** – Banco de dados NoSQL utilizado para armazenar de forma flexível os documentos de produtos, usuários e o histórico do sistema de trocas.
*   🛑 **Redis** – Banco de dados em memória (chave-valor) utilizado para cache de produtos de alta demanda e gerenciamento rápido de estados temporários de sessão.
